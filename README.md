# Beat Drift

Portrait rhythm lane dodger built with Flutter + Flame.

## 快速开始

1. 复制环境变量模板并填写：

```bash
cp .env.example .env
```

2. 拉取依赖并生成本地化代码：

```bash
flutter pub get
flutter gen-l10n
```

3. 运行：

```bash
flutter run
```

## 验证

```bash
flutter analyze
flutter test
flutter build apk --release
```
