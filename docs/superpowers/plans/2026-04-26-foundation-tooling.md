# Foundation & Tooling (Web) Implementation Plan
 
> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.
 
**Goal:** Scaffold a Next.js (App Router) + TypeScript (strict) + Tailwind foundation with CI-ready lint/format/typecheck and a minimal landing page.
 
**Architecture:** Keep the web app fully isolated at `/workspace/apps/web` with its own toolchain and a `src/`-based structure. Establish styling and quality gates early without introducing game logic yet.
 
**Tech Stack:** Next.js, React, TypeScript (strict), Tailwind CSS, ESLint, Prettier, Husky, lint-staged.
 
---

Set `/workspace/apps/web/tailwind.config.ts` to:
 
```ts
import type { Config } from "tailwindcss"
 
export default {
  content: ["./src/**/*.{ts,tsx}"],
  theme: {
    extend: {}
  },
  plugins: []
} satisfies Config
```

Create `/workspace/apps/web/src/styles/globals.css`:

html,

  height: 100%;
